1. copy the file tzmappings to the following directory: C:\Program Files\Java\jdk1.7.0\lib
2. amend the (system) PATH environment variable and add the path of the JDecafCompiler folder
3. execute the compiler using the command javadecaf from any folder within your system
4. execute your application

ex.	javadecaf JavaClass.java JavaClass2.java JavaDecafScript.jdf
	java JavaDecafScript
